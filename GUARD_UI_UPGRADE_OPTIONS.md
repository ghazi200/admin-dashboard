# 🚀 Guard-UI Upgrade Options

## Current Features (Baseline)
Based on codebase analysis, guard-ui currently supports:
- ✅ View shifts
- ✅ Create callouts
- ✅ Update availability
- ✅ Respond to callouts
- ✅ Clock in/out (time tracking)
- ✅ Start/end breaks
- ✅ Ask policy questions (AI RAG)
