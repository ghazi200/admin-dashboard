# Admin Dashboard Backend

Entry point: **`server.js`** (in this directory, not under `src/`).

**Start:**

```bash
npm start
```

or

```bash
node server.js
```

Copy `.env.example` to `.env` and set `DATABASE_URL`, `JWT_SECRET`, and for production `NODE_ENV=production` and `CORS_ORIGINS`. See `PRODUCTION_CHECKLIST.md` in the repo root.
